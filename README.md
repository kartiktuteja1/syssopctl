## Installation

### clone the repo first then:

```shell
cd syssopctl
mv  syssopctl.sh /usr/bin
sudo chmod +x syssopctl.sh
```

## commands available

./sysopctl --help
./sysopctl --version
./sysopctl service list
./sysopctl system load
./sysopctl disk usage
./sysopctl process monitor
./sysopctl logs analyze
./sysopctl backup /path/to/files
